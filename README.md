# cs50 journey
My personal solution, exercises, and experiments while following Harvard's CS50 course.

# note
all files require the cs50 library ("<cs50.h>").
you can install it by this official guide: [https://cs50.readthedocs.io/libraries/cs50/c/].

# the project included:
- crypto.c [This program encrypts messages using the Caesar cipher algorithm. The key is an integer. Each alphabetic character in the given text is shifted by that amount, based on the ASCII table.]
- - decrypto.c []
